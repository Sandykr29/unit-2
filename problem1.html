b tag

b tag is mainly known as "The Bring Attention To element". It is formerly know as the boldface element. While using this tag we can get the bold output of selected portion.

i tag

i tag is also know as "The Idiomatic Text element".In very simple way , if we put texts inbetween the i tag, the output we get are in the italic word form. It represents a range of text that is set off from the normal text for some reason, such as idiomatic text, technical terms, taxonomical designations, among others. Historically, these have been presented using italicized type, which is the original source of the i tag naming of this element.

u tag

u tag stands for "The Unarticulated Annotation (Underline) element ", it is used to render enclosed text with an underline. It represents a span of inline text which should be rendered in a way that indicates that it has a non-textual annotation. This is rendered by default as a simple solid underline, but may be altered using CSS.

select tag

select tag represents "The HTML Select element ", it is a control which provides a menu of options. Using these options we choose the appropriate value.

button tag

button tag provides us a kind of button that refers to either hyperlinking with other pages or starts another action on clicking.It is an interactive element activated by a user with a mouse, keyboard, finger, voice command, or other assistive technology. Once activated, it then performs a programmable action, such as submitting a form or opening a dialog. By default, HTML buttons are presented in a style resembling the platform the user agent runs on, but you can change buttons' appearance with CSS.

hr tag

hr tag stands for "The Thematic Break (Horizontal Rule) element " ,it draws a line that divides the content into parts.It represents a thematic break between paragraph-level elements: for example, a change of scene in a story, or a shift of topic within a section.

br tag

br tag stands for "The Line Break element". It produces a line break in text (carriage-return). It is useful for writing a poem or an address, where the division of lines is significant.
